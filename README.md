# eswc2018
The repository for ESWC2018 resource track

## 'Glue' Ontologies

This repository includes:
1. action.ttl
2. facilities.ttl
3. services.ttl
4. servies_facilities.ttl
5. services_targets.ttl
6. target_wordnet.ttl
7. targets.ttl
We call these 1 to 7 ontology files as 'glue' ontologies.
